# LeoUI

## Install

```bash
npm i -S @leooo/leoui
```

## Usage

```js
import Vue from 'vue'
import LeoUI from '@leooo/leoui'
import '@leooo/leoui/dist/leoui.min.css'
Vue.use(LeoUI)

new Vue({
  //...
})
```
