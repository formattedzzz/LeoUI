## Install

```bash
npm i -S @leooo/leoui
```

## Usage

```js
import Vue from 'vue'
import LeoUI from '@leooo/leoui'
import '@leooo/leoui/dist/leoui.min.css'
Vue.use(LeoUI)

new Vue({
  //...
})
```

## Development

```sh
git clone https://github.com/formattedzzz/LeoUI.git
cd LeoUI

npm install
# build complete leoui
npm run build
# build leoui demo
npm run dev
```
